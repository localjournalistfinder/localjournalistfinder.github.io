Create a Local Journalist Finder for Your City

Step 1: Fork the Repository

Start by creating your own copy of our GitHub repository:

  • Go to our GitHub repository
  • Click the "Fork" button in the top right corner
  • Name your repository "[your-city]journalistfinder.github.io"
  • Need help? See GitHub's guide on how to fork a repository.

Step 2: Update sources.json for Your City

The sources.json file contains the list of news outlets to track. You'll need to update it with outlets from your city:

{
  "newsapi": [
    "yourlocalexample.com",
    "citynewsoutlet.com",
    "localpaper.com"
  ],
  "rss": [
    "communityvoice.org",
    "neighborhoodnews.com"
  ]
}

To find local news outlets in your area:

  • Check Northwestern's State of Local News database
  • Look for news aggregator sites for your city
  • Search for "[your city] local news" to find outlets

Step 3: Get a NewsAPI Key

To fetch articles from news outlets, you'll need a NewsAPI key:

  • Sign up for a free account at NewsAPI.org
  • Copy your API key from the dashboard
  • Add it as a repository secret in GitHub:
    • Go to your repository Settings → Secrets and variables → Actions
    • Click "New repository secret"
    • Name: NEWSAPI_KEY
    • Value: Your NewsAPI key
